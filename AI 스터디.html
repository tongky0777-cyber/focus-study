📋선생님 대시보드

학생·반 관리, 진도와 숙제 입력, 시험 성적과 출결을 한 화면에서 관리합니다.

📱학생 클래스룸 앱

학생이 직접 가입해 숙제를 확인하고, 틀린 문제를 제출하는 휴대폰용 공간입니다.

✅오답 자동채점

학생이 답을 넣으면 객관식·단답이 자동 채점되고, 틀린 문제가 쌓입니다.

📊유형별 약점 분석

어떤 유형을 자주 틀리는지 자동으로 모아 학생별 약점을 한눈에 보여줍니다.

📝시험 성적 관리

기출·모의·데일리 점수를 입력하면 학생 홈에 점수와 추이가 표시됩니다.

🔔공지 · 질문 · 플래너

공지 전달, 학생 질문 답변, 학습 플래너까지 한 시스템 안에서 해결합니다.

2준비물 (꼭 챙겨오세요)
  • 노트북충전기까지. 휴대폰만으로는 어렵습니다.
  • 마우스파일을 옮기고 붙여넣는 작업이 많아 있으면 훨씬 편합니다.
  • 본인 이메일 계정로그인에 쓸 주로 쓰는 이메일과 비밀번호.
  • 스마트폰학생 화면을 직접 확인해볼 때 사용합니다.

  • GitHub 가입github.com 에서 무료 계정 만들기. 시스템 파일을 보관하는 곳입니다.
  • Vercel 가입vercel.com 에서 GitHub 계정으로 로그인. 시스템을 인터넷에 띄워줍니다.
  • Google 계정 준비학생 데이터 저장(Firebase)에 사용합니다.
  • 크롬 브라우저설치만 해두면 됩니다. 실습이 가장 잘 맞습니다.
